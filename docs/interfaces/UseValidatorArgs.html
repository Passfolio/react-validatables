Optional disabledWhile

disabledWhile?: BindingArrayDependencies | ReadonlyBinding<any>

If specified and the values of any of the specified bindings are truthy, the validator is disabled and so it will always result in "validity"

Optional disabledWhileUnmodified

disabledWhileUnmodified?: BindingArrayDependencies | ReadonlyBinding<any>

If any of the specified bindings are unmodified, the validator is disabled and so it will always result in "validity".

Note: if updating the modification state of these bindings without updating their values, you will need to directly call reset on this validator when ready to revalidate, since modification state changes on bindings don't trigger change callbacks.
